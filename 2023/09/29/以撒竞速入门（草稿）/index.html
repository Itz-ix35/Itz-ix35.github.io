以撒竞速入门（草稿）

发表于 2023-09-29 更新于 2024-06-02 分类于 【游戏】游戏相关
一些基本内容的介绍。

为消除歧义，本文讨论的竞速基于国内竞速比赛的体系，采用国内主要竞速玩家统一的竞速规则。

一、竞速的前提条件

  1. 拥有 Steam 正版以撒的结合：重生，并拥有全部 DLC（Afterbirth，Afterbirth+，Repentance）。
  2. 拥有一个全成就存档。
  3. 在创意工坊中订阅竞速 mod。

当需要竞速时，只需开启竞速 mod（名称为 !Bisai），然后进入游戏即可。

注意
  1. 一般来说，竞速时需要输入种子后进入游戏，这是为了确保重开后仍是同一个种子。（目前 mod 已经改善了这一点。）
  2. 开关 mod 后最好重启游戏，否则可能发生错误。

二、竞速规则

竞速流程：

  1. 玩家需要开启竞速 mod 并使用全成就存档，以普通模式进行竞速。
  2. 竞速应随机选择一个种子、一个角色、一个终点后进行，玩家不能提前得知有关种子的信息。
  3. 玩家输入种子，选择对应的角色和终点后开始计时。
  4. 玩家有三次游戏机会，每次死亡会消耗一次机会，死亡三次后机会耗尽。
  5. 当死亡三次机会耗尽、通过终点、或玩家主动投降时计时结束，结算成绩。

成绩结算：

  1. 玩家的成绩由层数和时间两者构成。层数即玩家在本次竞速中到达过的最远层数，时间即到达这一层时的最小时间。
  2. 按照层数从大到小为第一关键字，时间从小到大为第二关键字排序，排在前面的为胜者。若二者都相同则不分胜负。
  3. 时间按照真实时间（Real Time）计算。
层数计算
  • 通过终点，则层数为 99，时间即为最终 boss 血量归零的时刻。
  • 主线的八层（地下室 I 至血宫 II）的层数分别为 1 至 8。
  • 支线的八层（下水道 I 至尸宫 II）的层数分别为 2 至 9。
  • ??? 层（凹凸层）层数为 9。
  • 大教堂/阴间 层数为 10。
  • 玩具箱/暗室 层数为 11。
  • 由主线六层开启的陵墓 II 层数为 6.5，陵墓 II 回溯层数也是 6.5，和陵墓 II 算作同一层。
  • 回溯六层层数为 7，回溯五层层数为 8，以此类推，直到家的层数为 13。

注意事项：

  1. 不允许开启除竞速 mod 外的其他 mod，不允许在控制台输入有效代码。
  2. 虽然 mod 提供了暂停真实时间计时的方法，但原则上在竞速中不应使用，除非遇到特殊情况需要暂停。
  3. 不允许通过频繁在游戏内暂停达到特定目的（如持有混乱王冠的情况下正确拿取道具，或无伤躲避密集的弹幕）。
  4. 不允许主动重开游戏，只有死亡后才可以开始新的一局游戏。
  5. 不允许小退（即退回主菜单再进入游戏）。
  6. 不允许通过献祭到达暗室。
  7. 允许利用游戏 bug，但如果游戏过程中由于各种原因导致游戏崩溃，则根据具体情况处理，一切后果自负。
  8. 如果由于某些原因导致玩家返回到了之前的游戏状态（如游戏崩溃，或不小心重开游戏），则需要按照此前的原始路线重新进行一次游戏流程，不能修改探索顺序。

与其他项目规则的不同：

总的来说，国内竞速与 Speedrun 和 Racing+ 竞速区别非常大，但和 Daily Run 竞速十分接近。

与 Speedrun

Speedrun 翻译应该是速通，以撒的速通和其他游戏一样，要求以一切手段在尽可能短的时间内通关。与竞速的主要区别是：

  1. Speedrun 选择固定的种子，runner 提前知道种子的全部路线，而竞速中玩家不知道有关种子的信息。
  2. Speedrun 一般会使用献祭到达暗室，但这在竞速中被禁止。
  3. Speedrun 以对操作的极限优化为主，而竞速则主要追求期望最优的决策。

一般来说，你看得到的两分钟以内的都是 Speedrun，而竞速较少有 10 分钟以下的。

在 Speedrun 网站中有收录此项目的记录：https://www.speedrun.com/repentance#1_Char_Seeded

与 Racing+ 竞速

Racing+ 竞速是国外以撒圈统一使用的一套竞速规则，使用 Racing+ mod。与国内竞速的主要区别是：

  1. Racing+ 竞速的双方/多方的种子不是统一的，每个人的种子都不一样，而国内竞速的种子是统一的。
  2. 使用的 mod 不同，Racing+ mod 对于游戏本身内容有非常多的改动，而国内比赛 mod 对游戏本身内容改动比较小。

在 Speedrun 网站中有收录此项目的记录：https://www.speedrun.com/repentance#Racing

与 Daily Run 竞速

Daily Run 是以撒官方的每日挑战，分为竞速和刷分两个榜。其竞速与比赛竞速的主要区别是：

  1. 以游戏时间计时，而非真实时间；
  2. Daily Run 和比赛 mod 各有一些对游戏内容的微调，不尽相同。

许多竞速选手同时也热衷于完成每天的 Daily Run 竞速，然而作者并不喜欢打 Daily Run。

角色：

竞速中可以选择的角色包含几乎所有的角色（包括表里），除了以下角色：

  • 雅各和以扫（表双子）：由于该角色难度较高，同时许多玩家比较讨厌这一角色，所以在近期的竞速比赛中被 ban。
  • 堕化该隐（里该隐）：里该隐是一个非常吃随机的角色，即使是同一个种子也会产生完全不同的道具组合，被认为完全看运气，同时里该隐通常需要充分运营，导致游戏时间过长，因此被 ban。
  • 堕化拉撒路（里拉撒路）：里拉撒路通常需要大量运营，很难在可接受的时间内完成一局游戏，因此被 ban。
  • 堕化伊甸（里伊甸）：里伊甸的角色特性导致其玩法随机性极大，也是完全看运气。同时该角色难度很高，在竞速中较难通关（在一个有关道具池的 bug 被发现后，这一点就不成立了），因此被 ban。

曾经被 ban 过的角色还包括堕化遗骸（里骨）和堕化雅各（里双子），但现在都被放出来了。

许多角色有很多共性，但是有几个角色竞速玩法和其他角色有较大区别，将在之后介绍。

终点：

竞速 mod 提供的终点包括：

  • 大撒旦：以击杀超级撒旦（Mega Satan）作为目标。
    • 自带钥匙碎片 1，钥匙碎片 2，这会导致超级撒旦的金门会直接开启，注意这同时也会影响天使掉落和天使房转换率。
  • 百变怪：以击杀精神错乱（Delirium）为目标。
    • 该终点目前被弃用。
  • 见证者：以击杀见证者（Mother）为目标。
    • 从第二层开始，玩家被强制进行支线流程。
    • 所有支线层的门以及挑战强化妈心的门将直接开启，不需要任何条件。
  • 大霍恩：以击杀祸兽（The Beast）为目标。
    • 击杀妈腿后，会在妈腿房间中掉落一张 The Fool 卡牌，方便玩家回到初始房间开启通往陵墓 II 的门。
  • 羔羊：以击杀羔羊（The Lamb）为目标。
  • 小蓝人：以击杀小蓝人（???）为目标。
  • 凹凸：以击杀凹凸（Hush）为目标。

除百变怪外，其他目标均有可能作为竞速的终点。

一般来说 mod 会防止你走错路线，例如选择大霍恩终点时在第五层不会提供下陵墓的暗门，选择羔羊终点时不会提供全家福和通往教堂的光柱，这里不一一介绍。

三、竞速 mod 介绍

竞速时需要使用竞速 mod，其主要功能是提供真实时间计时，方便各游戏线路的进行（见"终点"部分），以及对游戏内容做出一些平衡性调整。

下面介绍一部分 mod 进行的平衡性调整：

  • 皇帝卡（The Emperor）的效果从 传送至当前层 boss 房 变为 传送至当前层随机房间。
  • 星象房道具池中，地球和水星的权重降低一半。
  • C 计划，TM 训练家被移除。
