    X01 Euterpe

    动画《罪恶王冠》第一集片头曲，由剧中音乐团体 EGOIST 主唱楪祈演唱，歌曲部分 CV 为 chelly。这首歌对我来说属于“第一次听就很震撼人心的歌”。

    楪祈一被造出来就是个工具人，但在剧情发展中她成功找到了自我的主体性，跳脱出别人的计划，最终成为一个独立的个体并为了大义献身。既脆弱又坚强，既理性又情绪化，这些对立的特质在对楪祈的塑造上表现地很成功。我不喜欢 GC 的主要剧情以及樱满集的形象，简单来说就是我觉得 GC 形似 EVA 却不神似（集同样形似真嗣却不神似），但是楪祈是我非常喜欢的一个女性形象，可以同样和 EVA 对比一下：明日香和绫波丽作为两个无比经典的女性形象，可能分别对不少人来说是阿尼玛似的存在，但是每个人心中的原型各有不同，对我而言，楪祈就是最符合我想象中”夏娃“形象的女性。为什么我不同样用“祈形似丽但不神似”来形容呢？因为后者的形象太复杂了，丽一个人几乎代表了整个“女性”——同时担当了母亲、妻子、少女等太多角色——祈顶多只是其中一个切片罢了，但是我觉得切得好。

    一码归一码，我还是比较喜欢 GC 的某些设定的。一个人心中的 void 一旦被毁，人也就跟着消亡——这是个很有意思的隐喻。void 代表了人的主体性，它表现为你最在意的东西、你的才能、你的记忆等等。这里就能引出一些有趣的话题，比如楪祈作为一个诞生之初不被认为有主体性的对象，其 void 确实威力巨大的大剑，这也照应了我上一段所说的她对自身主体性的追寻。再回到这首歌，结合 GC 的世界观和战争背景，一方面这显然是又一首反战的、祈望和平的歌曲；另一方面我们也可以将它看作一首后启示录的挽歌——Euterpe 是希腊神话中司抒情诗的女神，这首歌像是她在人类毁灭之后为其献上的一首长诗。这人类的灾难从何而来呢？如果不让真名来背锅的话，我们或许可以找到别的解释。古时就有天下“分久必合，合久必分”的说法，按照我的理解，GC 中如此强调人的主体性和异质性，也存在着政治上的隐喻——这种异质性导致了世界终究不能大同，试图把全人类变成橙汁这样的计划一定不可能——至少是以不流血的方式——实现。

    有空也聊聊 EVA 吧——一部我用喜欢和讨厌这样的词都无法形容的番，也许是借《残酷天使》之名？

    最后说一些题外话。24 年的夏天我与一些同学去新疆进行实践，当时刚看完 GC 倒没什么感触，不过在一周之内大巴车上循环了几百次这首 Euterpe 后却愈发喜欢上这首歌了。
