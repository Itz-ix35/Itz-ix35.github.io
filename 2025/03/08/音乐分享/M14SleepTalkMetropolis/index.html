M14 Sleep Talk Metropolis

个人认为 Mili 近年电子音乐探索方面的集大成者。这首歌在保留了一些 Mili 特色的同时，风格十分大众化与流行化，相信是新人入坑的一个好的选择。

现代社会中作为螺丝钉存在的渺小个体的迷思，这一直是文艺作品里屡见不鲜的主题。生活就像一列火车一样轰鸣着一刻不停地前进，我们的经历和体验像沿途的风景一样不断被甩向身后——我想借此谈谈列车这一意象。在我看来，列车本质上是单向流逝的时间，我们却将它投射到与时间有关的一切——记忆、生活，乃至整个世界。XanderGame 的《白色空间》除了化用 Omori 的情节外，我认为设计上最出色的一点就是将火车的实体与意象做了很好的融合，写出了很有名的 Create More Memories, And Keep This Train Moving，让当时的我觉得这比喻巧妙无比。这个意象在通俗文化中同样盛行，曾经看过的《歌手》现场里就有很多与列车有关的歌，像李健的《车站》，Tia 的《开往春天的地铁》，深究起来都属于这个意象的外延。

在我上面提到的几个作品中，这首 Sleep Talk Metropolis 在情感表达上无疑和《开往春天的地铁》最为相似（当然也颇有不同）。地铁在列车的基础上增加了“都市”和“通勤”的概念。如果长途火车让我们想起背井离乡，想起再不能见到的亲人，那么地铁就让我们想起了大城市里孤独的日复一日。资本主义的异化是一个社会性的现象，但我更喜欢的方式是将其分解成每一个个体的微观的问题，而 Sleep Talk Metropolis 也是这样做的——一个每天过着一样生活的社畜，幻想中也只是些世俗的愿望而已：想要旅行、想要房子、想要约会、想要孩子。然而这些也仅仅是遥不可及的梦境泄露下来的一点天光罢了。这首歌的歌词带有 momo 常用的堆砌意象的手法，但却非常直白，一看就懂；不同于其他 Mili 歌曲中深刻的内核带给人的思考，这首歌乍一看没什么但却细思极恐，仿佛将其中具体的细节遮蔽起来就是你我的生活。当然，如火车一样碾过身体的压抑的生活也并不是一无是处——那个由我们自身建构的遥远梦想就是引导自己最大的光——尽管它也许只存在于我们深睡时的梦境之中，或是日日早晚乘坐的那班地铁之上。
